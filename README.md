 📊 Previsão de Produção de Petróleo com Séries Temporais

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido no contexto da disciplina **Projeto Aplicado IV – Ciência de Dados (2026/01)**, com o objetivo de aplicar técnicas de análise de dados no setor de energia.

A proposta consiste em prever a **produção futura de petróleo** para apoiar decisões estratégicas de desinvestimento da Petrobras, utilizando técnicas de **modelagem de séries temporais**.

---

## 🎯 Objetivo

### Objetivo Geral
Descrever o comportamento histórico e prever a produção futura de óleo por meio da análise e modelagem de séries temporais.

### Objetivos Específicos
- Coletar e organizar dados históricos de produção de petróleo  
- Realizar análise exploratória dos dados (EDA)  
- Identificar padrões como tendência e sazonalidade  
- Aplicar modelos de séries temporais (ex: ARIMA)  
- Avaliar o desempenho dos modelos  

---

## 🛢️ Contexto do Problema

A negociação de campos de petróleo envolve decisões estratégicas de alto impacto financeiro. A capacidade de prever a produção futura de um campo é fundamental para determinar seu valor de mercado.

A utilização de técnicas de ciência de dados permite:
- Reduzir incertezas  
- Apoiar decisões estratégicas  
- Maximizar a rentabilidade em negociações  

---

## 📚 Referencial Teórico

O projeto baseia-se em conceitos de **análise de séries temporais**, incluindo:

- Tendência  
- Sazonalidade  
- Ciclos  
- Modelos ARIMA e variações  

Também considera abordagens modernas baseadas em **machine learning** para previsão de dados temporais.

---

## ⚙️ Pipeline da Solução

O projeto segue um pipeline estruturado de ciência de dados:

1. **Definição do problema**
2. **Coleta de dados**
3. **Tratamento e limpeza**
4. **Análise exploratória (EDA)**
5. **Modelagem (ARIMA e/ou outros modelos)**
6. **Avaliação dos resultados**
7. **Documentação e conclusão**
